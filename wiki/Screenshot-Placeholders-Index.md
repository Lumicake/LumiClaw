# Screenshot Placeholders Index

This page tracks placeholder IDs used across wiki pages.

## Naming Convention

- `SP-<AREA>-<NN>`
- image path: `images/SP-<AREA>-<NN>-<slug>.png`

## Captures to Produce

- `SP-HOME-01` Main app overview
- `SP-START-01..03` onboarding sequence
- `SP-ARCH-01..02` architecture visuals
- `SP-CHAT-01..03` DM/group/message rendering
- `SP-AM-01..03` agent mode and overlay
- `SP-QA-01..03` quick actions + bubble voice controls
- `SP-VOICE-01..03` voice input and playback states
- `SP-SET-01..03` settings, permissions, security
- `SP-TOOLS-01..02` tool history and risk prompt
- `SP-AUTO-01..02` automation UI
- `SP-DEPLOY-01..02` deployment outputs
- `SP-TS-01..02` troubleshooting examples

## Recommended Capture Specs

- Format: PNG
- Width: 1400-1800 px for full-window captures
- Crop: keep relevant UI context, avoid personal data
- Annotation: optional red boxes for critical controls
